# docker mcp catalog show

DescriptionDisplay catalog contentsUsage`docker mcp catalog show [name]`

## [Description](#description)

Display the contents of a catalog including all server definitions and metadata. If no name is provided, shows the Docker official catalog.

## [Options](#options)

OptionDefaultDescription`--format`Supported: "json", "yaml".

## [Examples](#examples)

# [Show Docker's official catalog](#show-dockers-official-catalog)

docker mcp catalog show

# [Show a specific catalog in JSON format](#show-a-specific-catalog-in-json-format)

docker mcp catalog show my-catalog --format=json
