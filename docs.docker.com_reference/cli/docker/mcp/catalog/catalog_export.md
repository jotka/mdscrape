# docker mcp catalog export

DescriptionExport a configured catalog to a fileUsage`docker mcp catalog export <catalog-name> <file-path>`

## [Description](#description)

Export a user-managed catalog to a file. This command only works with catalogs that have been imported or configured manually. The canonical Docker MCP catalog cannot be exported as it is managed by Docker.
