# docker mcp catalog update

DescriptionUpdate catalog(s) from remote sourcesUsage`docker mcp catalog update [name]`

## [Description](#description)

Update one or more catalogs by re-downloading from their original sources. If no name is provided, updates all catalogs that have remote sources.

## [Examples](#examples)

# [Update all catalogs](#update-all-catalogs)

docker mcp catalog update

# [Update specific catalog](#update-specific-catalog)

docker mcp catalog update team-servers
