# docker mcp catalog create

DescriptionCreate a new empty catalogUsage`docker mcp catalog create <name>`

## [Description](#description)

Create a new empty catalog for organizing custom MCP servers. The catalog will be stored locally and can be populated using 'docker mcp catalog add'.

## [Examples](#examples)

# [Create a new catalog for development servers](#create-a-new-catalog-for-development-servers)

docker mcp catalog create dev-servers

# [Create a catalog for production monitoring tools](#create-a-catalog-for-production-monitoring-tools)

docker mcp catalog create prod-monitoring
