# docker desktop start

DescriptionStart Docker DesktopUsage`docker desktop start [OPTIONS]`

> Note
> 
> `docker desktop start` doesn't work when executed via SSH on Windows due to a limitation in how WinCred stores credentials securely.

## [Options](#options)

OptionDefaultDescription`-d, --detach`Do not synchronously wait for the requested operation to complete.`--timeout`Terminate the running command after the specified timeout with a non-zero exit code. A value of zero (the default) or -1 means no timeout.
