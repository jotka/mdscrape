# docker container start

DescriptionStart one or more stopped containersUsage`docker container start [OPTIONS] CONTAINER [CONTAINER...]`Aliases

An alias is a short or memorable alternative for a longer command.

`docker start`

## [Description](#description)

Start one or more stopped containers

## [Options](#options)

OptionDefaultDescription`-a, --attach`Attach STDOUT/STDERR and forward signals`--checkpoint`experimental (daemon) Restore from this checkpoint`--checkpoint-dir`experimental (daemon) Use a custom checkpoint storage directory`--detach-keys`Override the key sequence for detaching a container`-i, --interactive`Attach container's STDIN

## [Examples](#examples)

```console
$ docker start my_container
```
