# docker scout integration delete

DescriptionDelete a new integration configurationUsage`docker scout integration delete INTEGRATION`

## [Description](#description)

The docker scout integration delete command deletes a new integration configuration for an organization.

## [Options](#options)

OptionDefaultDescription`--name`Name of integration configuration to delete`--org`Namespace of the Docker organization
