# docker scout integration configure

DescriptionConfigure or update a new integration configurationUsage`docker scout integration configure INTEGRATION`

## [Description](#description)

The docker scout integration configure command creates or updates a new integration configuration for an organization.

## [Options](#options)

OptionDefaultDescription`--name`Name of integration configuration to create`--org`Namespace of the Docker organization`--parameter`Integration parameters in the form of --parameter NAME=VALUE
