# docker compose attach

DescriptionAttach local standard input, output, and error streams to a service's running containerUsage`docker compose attach [OPTIONS] SERVICE`

## [Description](#description)

Attach local standard input, output, and error streams to a service's running container

## [Options](#options)

OptionDefaultDescription`--detach-keys`Override the key sequence for detaching from a container.`--index`index of the container if service has multiple replicas.`--no-stdin`Do not attach STDIN`--sig-proxy``true`Proxy all received signals to the process
