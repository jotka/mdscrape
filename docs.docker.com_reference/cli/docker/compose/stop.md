# docker compose stop

DescriptionStop servicesUsage`docker compose stop [OPTIONS] [SERVICE...]`

## [Description](#description)

Stops running containers without removing them. They can be started again with `docker compose start`.

## [Options](#options)

OptionDefaultDescription`-t, --timeout`Specify a shutdown timeout in seconds
