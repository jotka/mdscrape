# docker volume rm

DescriptionRemove one or more volumesUsage`docker volume rm [OPTIONS] VOLUME [VOLUME...]`Aliases

An alias is a short or memorable alternative for a longer command.

`docker volume remove`

## [Description](#description)

Remove one or more volumes. You can't remove a volume that's in use by a container.

## [Options](#options)

OptionDefaultDescription`-f, --force`API 1.25+ Force the removal of one or more volumes

## [Examples](#examples)

```console
$ docker volume rm hello

hello
```
